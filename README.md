# 💰 CatatDuit

**Catat pengeluaran, kenali keuanganmu.**

Aplikasi pencatat keuangan pribadi yang simpel, privat, dan bisa dipakai offline. Tidak ada server, tidak ada login, tidak ada langganan.

---

## ✨ Fitur

- 📊 **Dashboard lengkap** — Estimasi saldo, cashflow bulanan, uang bebas setelah kewajiban
- 💸 **Catat 3 jenis transaksi** — Uang masuk, keluar, dan nabung
- 🎯 **Goals nabung** — Buat target dan pantau progress keseluruhan
- 📋 **Reminder tagihan** — Catat cicilan & subscription rutin agar tidak lupa
- 📈 **Tren & grafik** — Lihat pola pengeluaran 12 bulan terakhir
- 📤 **Export & import CSV** — Backup data kapan saja
- 🔒 **100% privat** — Semua data tersimpan di perangkat kamu, tidak ada yang bisa mengaksesnya

---

## 🛒 Cara Mendapatkan

CatatDuit dijual via marketplace Indonesia.

Setelah pembelian dikonfirmasi, license key dikirim via chat marketplace.

> **Simpan license key kamu baik-baik.** Key bisa dipakai ulang jika ganti HP atau buka di perangkat lain.

---

## 📱 Cara Pakai

1. Buka link app di browser HP (Chrome direkomendasikan)
2. Input license key yang diterima dari seller
3. Isi nama dan saldo awal
4. Mulai catat keuangan

Bisa di-install sebagai PWA (tambahkan ke homescreen) agar terasa seperti app native.

---

## 🔐 Privasi

- Data **tidak pernah keluar dari perangkat kamu**
- Tidak ada server, tidak ada database, tidak ada akun
- Tidak ada iklan, tidak ada pelacak aktivitas
- Kode berjalan sepenuhnya di browser

> ⚠️ Karena data disimpan di browser, **hapus cache = data hilang**. Gunakan fitur Export Data secara berkala sebagai backup.

---

## 🛠 Tech Stack

- HTML · CSS · Vanilla JavaScript
- localStorage (no backend)
- Chart.js · Lucide Icons · Inter Font
- PWA (Service Worker + manifest)
